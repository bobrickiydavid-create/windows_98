# Projekt Zaliczeniowy: Windows 98 Web OS 💾

## Opis projektu
Projekt stanowi zaawansowaną rekonstrukcję i symulację klasycznego środowiska systemowego Windows 98, działającą w pełni w przeglądarce internetowej. Głównym celem aplikacji jest wierne odtworzenie unikalnego interfejsu retro przy użyciu współczesnych technologii webowych oraz stworzenie dynamicznego, wielozadaniowego ekosystemu. 

Projekt pozwala na przeanalizowanie asynchronicznego zarządzania wieloma komponentami interfejsu użytkownika w czasie rzeczywistym oraz integracji autorskiej logiki klienckiej z relacyjną bazą danych po stronie serwera.

## 🛠 Wykorzystane technologie
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** PHP
* **Baza danych:** MySQL

## 📌 Główne funkcjonalności
* **Autoryzacja:** Bezpieczny system rejestracji oraz logowania użytkowników z pełną obsługą i ochroną sesji.
* **Zarządzanie środowiskiem okienkowym:** Dynamiczny system wielozadaniowości umożliwiający swobodne przeciąganie okien (Drag & Drop), ich minimalizowanie, maksymalizowanie, zamykanie oraz automatyczne zarządzanie warstwami (z-index).
* **Zintegrowane aplikacje:** Zestaw wbudowanych programów i narzędzi systemowych działających niezależnie w architekturze piaskownicy (sandbox), naśladujących natywne działanie klasycznego systemu operacyjnego.
